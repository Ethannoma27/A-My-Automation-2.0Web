🗂️ 各模块职责说明：

模块 工具 功能说明
✅ 数据维护 Notion 手动或自动维护矿机信息（型号、价格、库存、图片）
✅ 静态站生成 GitHub + Netlify 自动将 Notion 内容生成 HTML 产品页，支持多页面、快速构建
✅ 页面托管 Netlify 高速 CDN，免费托管产品站，无需服务器成本
✅ 下单机制 表单 / JS 插件 提交客户基础信息 + 收货方式，匿名订单
✅ 支付方式 USDT钱包地址 / 网关 支持钱包扫码或网关 API 支付，可手动/自动验单
✅ 订单通知 Webhook + 邮件 / 飞书 / 微信机器人 订单提交后通知你处理，接入采购流程
✅ 管理后台 Notion（订单 + 客户库） 所有订单自动写入 Notion（含付款状态）

✅ 示例流程（订单路径）：

  1. 客户访问你的矿机站点（静态页面）
  2. 浏览型号 → 填写下单表单（或点击 USDT 支付链接）
  3. 页面触发 Webhook，将订单信息发送给你的 Serverless 后端
  4. 后端写入 Notion 订单数据库
  5. 向你发送提醒（邮件/微信）
  6. 记录客户地址 + 型号 + 支付情况
  7. 你后台确认后手动安排发货（走国内采购 + 空运 / 海运）
